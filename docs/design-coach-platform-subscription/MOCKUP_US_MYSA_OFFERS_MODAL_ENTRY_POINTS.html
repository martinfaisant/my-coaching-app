US-MYSA-OFFERS-02 — Même modale catalogue, trois contextes

Aujourd’hui ces flux appellent Checkout sans affichage des offres. Cible : le premier CTA « S’abonner / Continuer vers le paiement » ouvre la modale (US-01) ; la redirection Stripe n’a lieu qu’après choix d’une offre dans la modale. La page dédiée /dashboard/coach-platform-subscription conserve sa section offres pleine page (hors modale) — pas de régression fonctionnelle.

Critères d’acceptation transverses

  • Même contenu modale (offres, i18n, erreurs) quel que soit le déclencheur.
  • Fermeture modale : l’utilisateur reste sur le même URL / même vue qu’avant ouverture.
  • Après succès Stripe : comportement actuel (retour, vérification session, bandeaux) inchangé.
