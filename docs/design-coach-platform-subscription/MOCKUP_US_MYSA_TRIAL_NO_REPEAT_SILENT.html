US-MYSA-TRIAL-04 — Référence future : essai 2 sur nouvelle offre (hors livraison MVP)

PO : un coach ayant consommé l’essai campagne 1 pourrait être éligible à un essai distinct lié à une nouvelle offre / nouvelle campagne (ex. « Plateforme + encaissements »). Pas de UI dédiée aujourd’hui — le modèle d’éligibilité doit être par campagne ou par price_id, pas un blocage global « plus jamais d’essai ». Quand une future campagne le prévoit, le rendu redevient celui de US-MYSA-TRIAL-01 uniquement pour les offres concernées.
