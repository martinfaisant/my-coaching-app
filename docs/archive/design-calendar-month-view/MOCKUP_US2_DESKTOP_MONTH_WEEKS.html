US2 — Vue desktop (≥ md)

Empilement vertical de toutes les semaines (lun–dim) couvrant le mois sélectionné. Aucun titre du type « Il y a 3 semaines » ou « Semaine actuelle — … » entre les blocs. Chaque semaine = même carte synthèse (style semaine du milieu actuelle) + grille 7 jours mode détaillé (même densité / gap que la semaine centrale aujourd’hui).

Cas à couvrir

  • Mois avec 4, 5 ou 6 lignes de semaines.
  • Dernière semaine : atténuer les jours après le dernier jour du mois (ex. 1–7 juin si mai se termine un samedi).
  • Pas de variante « condensée » pour certaines semaines : toutes au format détaillé + carte synthèse (si aucune donnée, comportement actuel de la carte vide / masquée).
