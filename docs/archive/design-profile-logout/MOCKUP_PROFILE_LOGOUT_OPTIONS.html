Page Profil – Où placer le bouton Déconnexion ?

Contexte : la page affiche la card formulaire (Mes informations, Sports, etc.) puis en bas une zone avec « Supprimer mon compte ». La déconnexion n'existe aujourd'hui que dans le menu mobile (drawer).

Option A – Section « Session » au-dessus de Supprimer

Nouvelle section « Session » avec Déconnexion, puis le bloc Supprimer inchangé. Hiérarchie claire : quitter la session vs supprimer le compte.

Vous êtes connecté avec jean@exemple.fr.

Option B – Un seul bloc « Compte » (Déconnexion + Supprimer)

Un seul bloc en bas : titre « Compte », puis Déconnexion (bouton danger) et Supprimer mon compte (bouton danger). Les deux actions liées au compte au même endroit.

Compte

Option C – Déconnexion à côté du bouton Enregistrer

Dans la ligne « Mes informations », ajouter un second bouton « Déconnexion » (variant outline ou danger) à droite. Visible sans scroller.

Mes informations

Le bloc « Supprimer mon compte » reste en bas de la page comme aujourd'hui.
