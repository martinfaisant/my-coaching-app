Solution A — Tableau horaires + édition inline

Principe : Les horaires sont affichés dans un tableau lisible (7 lignes = 7 jours). L’utilisateur voit tout de suite « qui est ouvert quand ». Pour modifier un jour, il clique sur « Modifier » : une zone s’ouvre sous la ligne avec les plages (début–fin) et « + Ajouter une plage ».

Composants : Badge (type), Input/Textarea (adresse), Button primary/muted/danger. Tableau sémantique + détails dépliants (accordéon) par jour.
