Spécification

  • Condition : path === '/dashboard/profile' (pas startsWith, pour ne pas sélectionner la tuile Profil sur /dashboard/profile/offers).
  • État sélectionné : mêmes classes que les autres liens du menu : bg-palette-forest-dark text-white shadow-lg shadow-palette-forest-dark/20. Pas de bordure en état sélectionné.
  • Texte et chevron : en état sélectionné, nom en text-white, chevron en text-white/80 (ou équivalent). Avatar : garder ring-2 ring-white ; fond avatar peut rester ou passer en bg-white/30 pour cohérence.
  • Composant : components/Sidebar.tsx — deux blocs à modifier (athlète et coach), même logique.

Design system : tokens palette-forest-dark, shadow-palette-forest-dark/20. Référence : docs/DESIGN_SYSTEM.md, liens du menu dans Sidebar.tsx.
