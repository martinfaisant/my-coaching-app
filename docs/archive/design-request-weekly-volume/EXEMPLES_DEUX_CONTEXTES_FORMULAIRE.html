Où ajouter « Temps à allouer » et « Volumes » ?

Il existe deux endroits dans l’app où l’athlète peut remplir une demande (sports + besoin). Tu veux qu’on ajoute temps + volumes dans les deux, ou seulement dans le premier ?

Contexte 1

Formulaire dans la modale « Voir le détail » du coach

Parcours : Page « Trouver un coach » → Clic sur « Voir le détail » sur une carte coach → S’ouvre une grande modale (photo, bio, notes, langues). En dessous : grille d’offres (Suivi découverte, Suivi complet…). Quand l’athlète sélectionne une offre, le formulaire apparaît en dessous.

Contenu actuel du formulaire (dans cette modale) :

  • Prénom / Nom (si profil incomplet)
  • Sports pratiqués (tuiles)
  • Besoin de coaching (texte)
  • Bouton « Envoyer la demande »

→ C’est ici qu’on ajoute Temps à allouer + Volumes par sport (obligatoires). Fichier : FindCoachSection.tsx (CoachDetailModal).

Voir le mockup complet : MOCKUP_REQUEST_FORM.html

Contexte 2

Formulaire dans la modale « Choisir ce coach » (sans offre)

Parcours possible : Un bouton « Choisir ce coach » peut ouvrir une petite modale qui contient uniquement : Sports pratiqués + Besoin de coaching (sans choix d’offre). Fichier : RequestCoachButton.tsx.

Contenu actuel de cette petite modale :

  • Sports pratiqués (tuiles)
  • Besoin de coaching (texte)
  • Bouton « Envoyer la demande »

→ Aujourd’hui cette modale n’a pas de choix d’offre. Si on y ajoute temps + volumes, il faudra les rendre obligatoires ici aussi pour rester cohérent.

Dans l’app actuelle, le flux principal est le Contexte 1 (Voir le détail → choix offre → formulaire). Le Contexte 2 existe dans le code mais peut ne pas être affiché selon les écrans.

Question pour toi :

Veux-tu qu’on ajoute « Temps à allouer » et « Volumes par sport » dans les deux formulaires (Contexte 1 + Contexte 2), ou uniquement dans le Contexte 1 (modale détail coach avec offre) ?
