Transitions fonctionnelles couvertes par US3

  • Coach ouvre une séance Réalisée d'un athlète → modale s'ouvre, États A, C affichés selon données disponibles.
  • Coach ouvre une séance Non réalisée → État B (carte Réalisé masquée).
  • Lecture seule stricte : aucun bouton d'édition, aucune saisie. Fermeture via X uniquement.
  • Si l'athlète édite ses actual_* / feedback / commentaire pendant la session coach → la modale ne se met pas à jour automatiquement (rafraîchissement à la prochaine ouverture).
