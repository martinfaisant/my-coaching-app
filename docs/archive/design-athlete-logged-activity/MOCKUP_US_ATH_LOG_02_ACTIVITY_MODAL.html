Modale activité — layout coach + retour athlète

Reprend la modale coach (date, sports, titre, moment, bloc métriques stone-50, détails) + le bloc retour athlète en dessous. Une seule saisie temps / distance / allure / D+ — pas de segment Statut, pas de « Temps réalisé » / « Distance réalisée » en double.

Règle de fusion (PO validée)

  • Bloc métriques = même composant / mise en page que CoachWorkoutForm (icônes, grille, textarea détails dans le bloc).
  • Libellé du bloc : « Activité réalisée » (pas « Objectif de la séance »).
  • Champs = valeurs réalisées stockées directement (pas target_* + actual_*).
  • Sous le bloc stone-50 : WorkoutFeedbackSection + commentaire coach (comme capture athlète).
  • Exclu : onglets Planifié / Réalisé / Non réalisé ; WorkoutTargetActualCards.
